MySQL入门之（四）下载并安装MySQL数据库

2015-10-08
SQL

在浏览器中输入MySQL官方地址www.mysql.com，选择页签”Downloads (GA)” => MySQL Community Edition（MySQL社区版本）。选择”Downloads (GA)”：

选择”MySQL Community Edition”：

选择”MySQL Community Server (GPL)”：

在”General Available (GA) Release”页签中选择相应的操作系统，这里我们选择”Microsoft Windows”。

下载MySQL 32位或64位压缩包（大概210M左右），下载之前你可以通过”控制面板” => “系统和安全” => “系统” => “系统类型“中查看本机是32位还是64位。

查看本机操作系统位数：

这里我们无需注册MySQL账户，直接点击”No thanks, just start my downloads.”进行下载。

下载完成后将压缩包解压至”d:\Courses\MySQL\tools”，MySQL将被解压缩至”d:\Courses\MySQL\tools\mysql-5.6.{xx}-x64”目录中。为了使用方便我们可以将mysql-5.6.{xx}-x64目录修改为mysql。

这里推荐使用”ZIP”的版本而不是”Windows Installer”这个版本。当你不需要它的时候可以简单的删除这个目录就可以了，你也可以很方便的移动或重命名目录，或者在同一台物理机上安装多个MySQL的版本。

MySQL数据库中包含了：

  • 服务器端程序（mysqld）；
  • 客户端命令行工具（mysql）；
  • 工具类：数据库管理工具（mysqladmin），数据备份/还原（mysqldump）及其他相关工具；
  • 客户端库：利用这些库你可以开发自己的客户端工具。
