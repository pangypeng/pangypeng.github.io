MySQL入门之（七）启动客户端

2015-10-08
SQL

回想一下MySQL所采用的客户端/服务器架构，一旦服务器启动完成，多个客户端就可以同时连接到数据库了，客户端即可以与服务器在同一台设备上，也可以通过网络远程连接至服务器。

在登录到MySQL服务器之前需要提供用户名和密码，默认情况下MySQL自动会创建没有密码的”root”用户（超级管理员）。

现在使用”root”用户启动一个客户端程序，首先，需要确认服务器端程序已经正常运行（如果服务器程序已经关闭的情况下需要重新启动）。

启动客户端程序

启动CMD控制台窗口并运行client客户端程序。 — 将目录切换至< MYSQL_HOME >\bin — 这里我假设MySQL已经安装至”d:\Courses\MySQL\tools\mysql”

Prompt> d:                                — 切换当前工作目录
Prompt> cd \Courses\MySQL\tools\mysql\bin  — 切换至MySQL\bin目录

— 使用”root”用户启动客户端程序

d:\Courses\MySQL\mysql\bin> mysql -u root
Welcome to the MySQL monitor.  Commands end with ; or \g.
Your MySQL connection id is 1
Server version: 5.6.14-community MySQL Community Server (GPL)
Type 'help;' or '\h' for help. Type '\c' to clear the current input statement.

— 此时已经进入MySQL命令行模式，提示信息为”mysql>”
— 现在你可以在这里输入SQL命令了

mysql>

在输入命令前先来查看一下MySQL的状态，这里我们输入”status”命令：

mysql> status
————–
mysql  Ver 14.14 Distrib 5.6.14, for Win64 (x86_64)
Current user:          root@localhost
Server version:        5.6.14 MySQL Community Server (GPL)
Protocol version:      10
Connection:            localhost via TCP/IP
TCP port:              8888
